Елементи HTML-форми

  • Тег form - блочний
  • Тег label - рядковий
  • Теги input, textarea, button - блочно-рядкові

input - багатофункціональний елемент вводу даних користувача

  • Атрибут type - тип елементу вводу даних (Усі типи https://www.w3schools.com/html/html_form_input_types.asp)
  • Атрибут name - ім'я поля (даних)
  • Атрибут value - значення поля (дані)
  • Атрибут placeholder - підпис поля
  • Атрибут autocomplete - дозволяє (за замовченням) / забороняє (значення off) автозаповнення поля
  • Атрибут required - обо'язково для заповнення
  • Атрибут tabindex - керує навігацією за допомогою TAB
  • Атрибут maxlength - максимальна кількість символів доступних для вводу
  • Атрибут minlength - вимога до мінімальної кількості символів. Працює в парі з required
  • Атрибут checked (для деяких типів) - вибраний за замовченням
  • Атрибут disabled - поле перестає бути доступним для користування, данні передані не будуть
  • Атрибут pattern - можливість вказувати регулярні вирази
