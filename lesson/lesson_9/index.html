Б.Е.М

Створюємо окремі незалежні частини, потім складаємо сторінку як конструктор та перевикористовуємо готові блоки.

Будь-який БЕМ-Блок стає компонентом.

Будь-який компонент обов'язково має бути побудований як БЕМ-Блок

НЕЗАЛЕЖНИЙ, САМОДОСТАТНІЙ об'єкт. НЕ має впливати на оточення (сусідні об'єкти).

І'мя класу БЕМ-блоку:

  1. І'мя класу має відповідати змісту або конструкції БЕМ-блоку. Перевірочне питання - "Що це?
  2. Англійска мова (Латиниця)
  3. Нижній регістр
  4. Якщо слів декілька, розділяємо дефісом (-).
  5. При формуванні імені класу не має бути спроби ієрархії

Дозволяється вкладення будь якого рівня та будь-якої кількості БЕМ-блоків один в одного.

БЕМ-Елемент

ЗАЛЕЖНИЙ об'єкт від СВОГО БЕМ-блоку

ЗАВЖДИ знаходиться всередені СВОГО БЕМ-блоку (вкладений)

МОЖЕ впливати на оточення (сусідні об'єкти).

І'мя класу БЕМ-елементу:

  1. ЗАВЖДИ починається з імені першого в ієрархії батьківського БЕМ-блоку і відокремлюється двома підкресленнями (__).
  2. І'мя має відповідати змісту або конструкції блоку. Перевірочне питання - "Що це?"
  3. Англійска мова (Латиниця)
  4. Нижній регістр
  5. Якщо слів декілька, розділяємо дефісом (-).
  6. При формуванні імені не має бути ієрархії.

БЕМ-елементи можна вкладати в інші БЕМ-елементи свого БЕМ-блоку

БЕМ-Модифікатор

Дозволяє вносити відмінності для БЕМ-блоків або БЕМ-елементів МОЖЕ впливати на оточення (сусідні об'єкти)

І'мя класу БЕМ-модифікатора:

  1. ДОДАЄТЬСЯ до існуючого БЕМ-класу блоку або елементу.
  2. ЗАВЖДИ починається з існуючого БЕМ-класу блоку або елементу і відокремлюється подвійним дфісом (--)
  3. І'мя має відповідати модифікаціям. Перевірочне питання - "Який? Що змінено?"
  4. Англійска мова (Латиниця)
  5. Нижній регістр
  6. Якщо слів декілька, розділяємо дефісом (-).
  7. При формуванні імені не має бути ієрархії.

БЕМ-Мікс

Після того як ви опануєте блок елемент модифікатор, саме час використовувати мікси. Мікс - це одночасне використання і БЕМ-блоку і БЕМ-елементу для одного об'єкту.

Переваги міксу:

  1. вивільнення зрозумілих слів для нових елементів
  2. можливість викорістання CSS-властивостей, які впливають на оточення, фактично для БЕМ-блоку (БЕМ-елемент в ролі модифікатора)
  3. паралельне відокремлення певної структури в БЕМ-блок

Часті слова

  1. container - обмежуючий контейнер (block__container)
  2. body, content - головна частина блоку (block__body)
  3. row - рядок
  4. column - колонка
  5. link - посилання
  6. title - заголовок
  7. sub-title - підзаголовок
  8. label - ще заголовок
  9. text - текст
  10. list - список
  11. items - багато якихось елементів
  12. item - будь-що
  13. menu
  14. block
  15. button
  16. picture, image, img
  17. section
  18. article
  19. unit
  20. actions, action
  21. value
  22. search
  23. form
  24. page
  25. box
